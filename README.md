# Not To Listen — CMS 423: Freedom of Speech

A school assignment for **CMS 423: Freedom of Speech** at the **University of Southern Maine**.

This project is an interactive single-page web tool exploring broadcast regulation and free speech frameworks, with a focus on the *Captive Audience Doctrine* and the regulatory landscape around indecency on the airwaves.

## Live Site

[https://jamacon36.github.io/CMS-423-Not-To-Listen/](https://jamacon36.github.io/CMS-423-Not-To-Listen/)

## About

The site uses an interactive honeycomb/hexagon diagram to visualize regulatory and analytical frameworks. All content is editable directly in the browser via `contenteditable` fields.

## Setup

No build tools or dependencies required.

1. Clone the repository:
   ```
   git clone https://github.com/jamacon36/CMS-423-Not-To-Listen.git
   ```
2. Open `index.html` in any modern web browser.

That's it — no installs, no build step.

## Tech Stack

- Pure HTML, CSS, and JavaScript (single file)
- Inline SVG for the hexagon diagram
- Google Fonts (Playfair Display, Source Serif 4) via CDN
